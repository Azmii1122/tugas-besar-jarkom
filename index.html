Perangkat Jaringan

Pembahasan mendalam mengenai hardware vital dalam infrastruktur IT.

1. Router

Router berfungsi menghubungkan dua atau lebih jaringan yang berbeda (misalnya LAN ke WAN/Internet). Ia bekerja pada Layer 3 (Network) model OSI dan menggunakan IP Address untuk menentukan rute terbaik (routing) bagi paket data.

2. Switch (Manageable)

Switch menghubungkan perangkat dalam satu segmen jaringan lokal. Berbeda dengan Hub, Switch cerdas karena menyimpan tabel MAC Address. Ia bekerja di Layer 2 (Data Link), mencegah collision data agar lalu lintas jaringan lebih efisien.

3. Wireless Access Point

Access Point (AP) bertindak sebagai pusat pemancar sinyal radio untuk perangkat nirkabel. Ia mengubah sinyal data kabel (wired) menjadi gelombang radio (wireless), memungkinkan mobilitas user dalam jaringan menggunakan standar IEEE 802.11.
